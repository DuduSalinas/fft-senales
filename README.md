# Simulación y análisis de señales con Fourier (FFT/STFT) en MATLAB y Python

## Objetivo
Analizar señales en tiempo y frecuencia aplicando FFT y STFT. Demostrar propiedades: linealidad, desplazamiento temporal y escalamiento, y comparar ventanas.

## Estructura
- `src/python/analisis_fft.py`: Script Python (FFT, ventanas, padding, propiedades).
- `src/python/analisis_fft_stft.ipynb`: Notebook Jupyter (FFT + STFT).
- `src/matlab/analisis_fft.m`: Script MATLAB (FFT y propiedades).
- `src/matlab/analisis_stft.m`: Script MATLAB (STFT y ventanas).
- `figures/`: Gráficas generadas automáticamente.
- `docs/presentacion_fft.pptx`: Plantilla PowerPoint editable.
- `docs/canva_texto_base.md`: Bloques de texto para pegar en Canva.
- `docs/informe_fft.md`: Guía del informe técnico.

## Requisitos
- Python 3.10+ (con NumPy, SciPy, Matplotlib, Jupyter).
- MATLAB (Online o escritorio).
